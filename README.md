# GS-Front-End
KESSL — Monitoramento Inteligente de Lixo Espacial
 
O KESSL é um projeto que apresenta uma plataforma de monitoramento de detritos orbitais, alertando sobre riscos de colisão com satélites ativos.
Sobre o Projeto:
O lixo espacial é um dos maiores problemas da exploração espacial moderna.
São mais de 27.000 fragmentos rastreados orbitando a Terra a 28.000 km/h, ameaçando satélites e missões tripuladas.
O KESSL propõe um sistema inteligente para monitorar esses detritos e prevenir colisões em tempo real.
 
Estrutura da Landing Page
A página é composta por 6 seções obrigatórias:
 
Problema — apresenta o problema do lixo espacial e seus impactos
Tecnologia — mostra as tecnologias utilizadas na solução
Objetivos — descreve os principais objetivos do projeto
Público-alvo — informa quais empresas e organizações serão impactadas
Benefícios — apresenta os benefícios oferecidos pela plataforma
Aplicação — explica como a solução funciona na prática
 
Tecnologias Utilizadas
 
HTML5
CSS3 (Flexbox, variáveis CSS, reset CSS)
Google Fonts (Orbitron e Poppins)
